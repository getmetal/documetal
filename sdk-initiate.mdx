## Instantiating
You can obtain the the following keys in the [Getting Started](/create-app) guide.

- `apiKey`
- `clientId`
- `appId`

```
import MetalSDK from '@getmetal/metal-sdk';

const metal = new MetalSDK(
  'api-key',
  'client-id',
  'app-id',
);
```
