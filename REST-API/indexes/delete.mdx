---
title: "Delete Index"
api: "DELETE https://api.getmetal.io/v1/indexes/{indexId}"
description: "This endpoint deletes an index and all documents within that index."
---

### Auth Headers

<ParamField required header="x-metal-api-key" type="string">
  An API key for your org.
</ParamField>

<ParamField required header="x-metal-client-id" type="string">
  A Client ID for your org.
</ParamField>

### Params

<ParamField required path="indexId" type="string">
  The Index ID
</ParamField>

### Response

<ResponseField name="accepted" type="boolean">Whether the deletion was successful or not.</ResponseField>

<RequestExample>
```bash Example Request
curl --location --request DELETE 'https://api.getmetal.io/v1/indexes/{indexId}' \
--header 'Content-Type: application/json' \
--header 'x-metal-api-key: <api-key>' \
--header 'x-metal-client-id: <client-id>' \
```

</RequestExample>

<ResponseExample>

```json Response
{
  "accepted": true
}
```

</ResponseExample>
