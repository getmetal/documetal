---
title: "Post Memory"
api: "POST /sessions/{sessionId}/memory"
description: "This endpoint deletes a single memory."
---

### Params

<ParamField required path="sessionId" type="string">
  The Session ID
</ParamField>

### Response

<ResponseField name="data" type="App Object">
  <Expandable title="properties">
    <ResponseField name="id" type="string">
      Id of the memory
    </ResponseField>

    <ResponseField name="name" type="string">
      Name of the memory
    </ResponseField>

  </Expandable>
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request POST '/sessions/{sessionId}/memory"' \
--header 'Content-Type: application/json' \
```

</RequestExample>

<ResponseExample>

```json Response
{
  "data": {
    "id": "1",
    "name": "Ozzy's memory"
  }
}
```

</ResponseExample>
