---
title: "Typescript SDK"
description: "Metal's Typescript SDK."
---

## Installing
### npm

```bash
npm i --save @getmetal/metal-sdk
```

### yarn

```bash
yarn add @getmetal/metal-sdk
```

## Instantiating

```
import MetalSDK from '@getmetal/metal-sdk';

const metal = new MetalSDK(
  'api-key',
  'client-id',
  'app-id',
);
```


## Indexing Data
```javascript
const ingestedData = await metal.index('Text data to ingest');
```

## Searching Data
```javascript
const searchResults = await metal.search('Term to search');
```
