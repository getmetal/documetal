## Installation

```bash
npm install --save @getmetal/metal-sdk
```

## App Setup

- `apiKey` - The API Key for your Org. [Create here](https://app.getmetal.io/settings/organization).
- `clientId` - The Client ID for your Org. [Create here](https://app.getmetal.io/settings/organization).
- `appId` - The ID of the app you want to connect to.

```typescript
import MetalSDK from "@getmetal/metal-sdk";

const metal = new MetalSDK("apiKey", "clientId", "appId");
```
