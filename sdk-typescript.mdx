<Note>You'll need an [API Key](/misc-get-keys) to use the SDK.</Note>
## Installation

```bash
npm install --save @getmetal/metal-sdk
```

## App Setup

| Param      | Type   | Description                                                |
| ---------- | ------ | ---------------------------------------------------------- |
| `apiKey`   | string | The API Key for your Org. [Create here](/misc-get-keys).   |
| `clientId` | string | The Client ID for your Org. [Create here](/misc-get-keys). |
| `appId`    | string | The ID of the app you want to connect to.                  |

```typescript
import MetalSDK from "@getmetal/metal-sdk";

const metal = new MetalSDK("apiKey", "clientId", "appId");
```

## Indexing

| Param         | Type     | Description                                      |
| ------------- | -------- | ------------------------------------------------ |
| `embedding`   | number[] | Raw embeddings to be indexed.                    |
| `imageBase64` | string   | A Base64 image to be vectorized and indexed.     |
| `imageUrl`    | string   | A URL for an image to be vectorized and indexed. |
| `text`        | string   | The text to be vectorized and indexed.           |

```typescript
await metal.index({ text: "text-to-index" });
```

## Searching

| Param         | Type     | Description                                       |
| ------------- | -------- | ------------------------------------------------- |
| `embedding`   | number[] | Raw embeddings to be searched.                    |
| `imageBase64` | string   | A Base64 image to be vectorized and searched.     |
| `imageUrl`    | string   | A URL for an image to be vectorized and searched. |
| `text`        | string   | The text to be vectorized and searched.           |

```typescript
const results = await metal.search({ text: "term-to-search" });
```

## Tuning

| Param   | Type   | Description                                                |
| ------- | ------ | ---------------------------------------------------------- |
| `idA`   | string | The ID of the first embedding..                            |
| `idB`   | string | The ID of the second embedding.                            |
| `score` | string | The "closeness score" of the two items. Either '-1' or '1' |

```typescript
await metal.tune(idA, idB, score);
```
