<Note>You'll need an [API Key](/misc-get-keys) to use the SDK.</Note>
## Installation

```bash
npm install --save @getmetal/metal-sdk
```

## App Setup

| Param      | Type   | Description                                                |
| ---------- | ------ | ---------------------------------------------------------- |
| `apiKey`   | string | The API Key for your Org. [Create here](/misc-get-keys).   |
| `clientId` | string | The Client ID for your Org. [Create here](/misc-get-keys). |
| `appId`    | string | The ID of the app you want to connect to.                  |

```typescript
import MetalSDK from "@getmetal/metal-sdk";

const metal = new MetalSDK("apiKey", "clientId", "appId");
```

## Indexing

| Param  | Type   | Description                            |
| ------ | ------ | -------------------------------------- |
| `text` | string | The text to be vectorized and indexed. |

```typescript
await metal.index("text-to-index");
```

## Searching

| Param  | Type   | Description                        |
| ------ | ------ | ---------------------------------- |
| `term` | string | The term for the search operation. |

```typescript
const results = await metal.index("term-to-search");
```

## Tuning

| Param   | Type   | Description                                                |
| ------- | ------ | ---------------------------------------------------------- |
| `idA`   | string | The ID of the first embedding..                            |
| `idB`   | string | The ID of the second embedding.                            |
| `score` | string | The "closeness score" of the two items. Either '-1' or '1' |

```typescript
await metal.tune(idA, idB, score);
```
