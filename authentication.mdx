description: "Metal API Keys to use our API & SDKs."
---

<Note>
  To create an [API Key](/misc-get-keys) in the App, [read
  this](/misc-get-keys).
</Note>

With your API Key, you can use authenticate by populating the following headers:

| Title     | Header              |
| --------- | ------------------- |
| API Key   | `x-metal-api-key`   |
| Client ID | `x-metal-client-id` |
