<Note>You'll need an [API Key](/misc-get-keys) to use the SDK.</Note>
## Installation

```bash
pip install metal-sdk
```

## App Setup

| Param       | Type   | Description                                                |
| ----------- | ------ | ---------------------------------------------------------- |
| `api_key`   | string | The API Key for your Org. [Create here](/misc-get-keys).   |
| `client_id` | string | The Client ID for your Org. [Create here](/misc-get-keys). |
| `app_id`    | string | The ID of the app you want to connect to.                  |

```python
from metal_sdk.Metal import Metal

metal = MetalSDK("api_key", "client_id", "app_id");
```

## Indexing

| Param         | Type     | Description                                      |
| ------------- | -------- | ------------------------------------------------ |
| `embedding`   | number[] | Raw embeddings to be indexed.                    |
| `imageBase64` | string   | A Base64 image to be vectorized and indexed.     |
| `imageUrl`    | string   | A URL for an image to be vectorized and indexed. |
| `text`        | string   | The text to be vectorized and indexed.           |

```python
metal.index({ text: "text-to-index" });
```

## Searching

| Param         | Type     | Description                                       |
| ------------- | -------- | ------------------------------------------------- |
| `embedding`   | number[] | Raw embeddings to be searched.                    |
| `imageBase64` | string   | A Base64 image to be vectorized and searched.     |
| `imageUrl`    | string   | A URL for an image to be vectorized and searched. |
| `text`        | string   | The text to be vectorized and searched.           |

```python
results = metal.search({ text: "term-to-search" });
```

## Tuning

| Param   | Type             | Description                                                |
| ------- | ---------------- | ---------------------------------------------------------- |
| `id_a`  | string           | The ID of the first embedding..                            |
| `id_b`  | string           | The ID of the second embedding.                            |
| `label` | number[-1, 0, 1] | The "closeness score" of the two items. Either -1, 0, or 1 |

```python
metal.tune('id_a', 'id_b', 1);
```
