## 2. Embed text into Metal
The first step is to embed your text into Metal. This will allow you to compare your text to other text in the index.

```python

from metal_sdk.metal import Metal

metal = Metal(
  'pk_1234567890',  # api-key
  'ci_1234567890',  # client-id
  'index_1234567890', # index-id
);



embedding_document = metal.index({
  "text": "A rocket flying towards mars.",
  "metadata": {
    "category": "space",
  },
});

print(embedding_document);

```

## 3.Run a Semantic Search
Next, we'll want to run a semantic search against the index. This will allow us to find similar text to the text we just embedded.

```python

from metal_sdk.metal import Metal

metal = Metal(
  'pk_1234567890',  # api-key
  'ci_1234567890',  # client-id
  'index_1234567890', # index-id
);


results = await metal.search({
  "text": "space travel",
});

print(results);

```
