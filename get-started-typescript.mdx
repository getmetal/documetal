## Installing

### npm

```bash
npm i --save @getmetal/metal-sdk
```

### yarn

```bash
yarn add @getmetal/metal-sdk
```

### Simple Example

```TypeScript
import Metal from '@getmetal/metal-sdk';

// Instantiate your Metal App
const metal = new Metal(
  'api-key',
  'client-id',
  'app-id',
);

// Index text into your App
await metal.index({ text: 'Tree' });
await metal.index({ text: 'Food' });
await metal.index({ text: 'Truck' });

// Run Search operations on your index
const result = await metal.search({ text: 'car' });
console.log(result) // 'Truck'
```
