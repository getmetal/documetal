## Retrieval


### 1. Install SDK

<CodeGroup>

```bash Python
pip install metal-sdk
```
```bash Node
npm install @getmetal/metal-sdk
```
</CodeGroup>

### 2. Initialize Metal Client

<CodeGroup>

```python Python
from metal_sdk.metal import Metal

metal = Metal(
  "pk_1234567890",
  "ci_1234567890",
  "index_1234567890"
)
```
```javascript Node
import { Metal } from "@getmetal/metal-sdk";

const metal = new Metal(
  'pk_1234567890', // api-key
  'ci_1234567890', // client-id
  '1234567890', // index-id
);
```
</CodeGroup>

### 3. Embed your Documents

<CodeGroup>
```python Python
embedded_document = metal.index({
  "text": "A rocket flying towards mars.",
  "metadata": {
    "category": "space",
  },
})

print(embedded_document)
```
```javascript Node
(async () => {
const embeddingDocument = await metal.index({
text: 'A rocket flying towards mars.',
metadata: {
  category: 'space',
},
});

console.log(embeddingDocument);
})();
```
```python Web UI
# 1. Go to https://app.getmetal.io
# 2. Navigate to your index dashboard
# 3. Click "+ Import"
```
</CodeGroup>

### 4. Query your Documents

<CodeGroup>
```python Python
results = metal.search({
  "text": "space travel",
  "filters": {"and": [{"field": "category", "value": "space", "operator": "eq"}]},
  "limit": 3
})

print(results)
```

```javascript Node
(async () => {
const results = await metal.search({
text: 'space travel',
filters: { and: [{ field: 'category', value: 'space', operator: 'eq' }] }
});

console.log(results);
})();
```

```python Web UI
# 1. Go to https://app.getmetal.io
# 2. Navigate to your index browse page
# 3. Search for a term in the input box
```
</CodeGroup>

### 5. Filter your Results

Filtering allows you to narrow down your search results by a specific field.

`and` and `or` clauses are supported. Every `and` clause has to be satisfied and at least one `or` clause has to be satisfied to match a document. The operators supported are `eq`, `gt`, `gte`, `lt` and `lte`. A simple example:

<CodeGroup>

```python Python
results = metal.search({
  "text": "space travel",
  "filters": {
    "and": [
      {
        "field": "planet",
        "value": "mars",
        "operator": "eq"
      },
      {
        "or": [
          {
            "field": "gravity",
            "value": 0.1,
            "operator": "gte"
          },
          {
              "field": "gravity",
              "value": 0.9,
              "operator": "lte"
          }
        ]
      }
    ]
  }
})
print(results)
```
```javascript Node
(async () => {
    const results = await metal.search({
        text: 'space travel',
        filters: {
            and: [
                {
                    field: 'planet',
                    value: 'mars',
                    operator: 'eq'
                },
                {
                    or: [
                        {
                            field: 'gravity',
                            value: 0.1,
                            operator: 'gte'
                        },
                        {
                            field: 'gravity',
                            value: 0.9,
                            operator: 'lte'
                        }
                    ]
                }
            ]
        }
    });

    console.log(results);
})();
```

</CodeGroup>

## Memory

### 6. Initialize Motorhead Client

<CodeGroup>

```python Python
from metal_sdk.motorhead import Motorhead

motor = Motorhead({
  "api_key": "pk_1234567890",  # api-key
  "client_id": "ci_1234567890", # client-id
})
```
```javascript Node
import { Motorhead } from "@getmetal/metal-sdk";

const motor = new Motorhead({
  apiKey: "pk_1234567890", // api-key
  clientId: "ci_1234567890", // client-id
});
```
</CodeGroup>

### 7. Add a Memory Session
Add a Memory Session to track LLM sessions for your users.

<CodeGroup>
```python Python
mem = motor.add_memory(
  "my-session-id",
  {
    "messages": [
      { "role": "Human", "content": "hello, ai" },
      { "role": "AI", "content": "hello, human" },
    ],
  },
)
print(mem)

```
```javascript Node
(async () => {
  const mem = await motor.addMemory("my-session-id", {
    messages: [
      { role: "Human", content: "hello, ai" },
      { role: "AI", content: "hello, human" },
    ],
  });

  console.log(mem);
})();
```
</CodeGroup>

### 8. Get a Memory Session
We'll then want to allow you to fetch your memory session so that you can provide the LLM with historical context.

<CodeGroup>
```python Python
mem = motor.get_memory("my-session-id")
print(mem)
```
```javascript Node
(async () => {
  const session = await motor.getMemory("my-session-id");
  console.log(session);
})();
```
</CodeGroup>
