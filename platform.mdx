## Nouns

Critical resources of the Metal Platform.

| Resource | Description                                           |
| -------- | ----------------------------------------------------- |
| App      | A resource that represents your specific ML use-case. |
| Index    | An indexed vector database of your _Embeddings_       |
| Org      | An organization                                       |
| Tuning   | A mechanism to fine-tune your _App_ programatically.  |
