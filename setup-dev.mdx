## Installing
### npm

```bash
npm i --save @getmetal/metal-sdk
```

### yarn

```bash
yarn add @getmetal/metal-sdk
```

## SDK Usage

```
import MetalSDK from '@getmetal/metal-sdk';

const metal = new MetalSDK(
  'api-key',
  'client-id',
  'app-id',
);


await metal.index('Tree');
await metal.index('Food');
await metal.index('Truck');

const result = await metal.search('car');
console.log(result) // 'Truck'

```
