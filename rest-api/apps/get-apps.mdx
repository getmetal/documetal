---
title: "Get All Apps"
api: "GET https://api.getmetal.io/v1/apps"
description: "This endpoint retrieves a list of all apps for the specified organization"
---

## Auth Headers
<ParamField required header="x-metal-api-key" type="string">
  An API key for your org.
</ParamField>
<ParamField required header="x-metal-client-id" type="string">
  A Client ID for your org.
</ParamField>

## Response

<ResponseField name="data" type="App Object">
  <Expandable title="properties">
    <ResponseField name="id" type="string">
      Id of the app.
    </ResponseField>
    <ResponseField name="name" type="string">
  Name of the app.
</ResponseField>

<ResponseField name="indexes" type="array">
  An array indicating the connected indexes for the app. Empty if there are no connected indexes.
</ResponseField>

</Expandable>
</ResponseField>

<RequestExample>
```bash
curl --location --request GET 'https://api.getmetal.io/v1/apps' \
--header 'Content-Type: application/json' \
--header 'x-metal-api-key: <api-key>' \
--header 'x-metal-client-id: <client-id>'
```
</RequestExample>

<ResponseExample>
```json
{
	"data": [
        {
		  "id": "6535126922af1b89d6d22ffa",
		  "name": "my metal app",
		  "indexes": []
	    },
        //... other app objects
    ]
}

```
</ResponseExample>
