---
title: "Delete Data Entity"
api: "DELETE https://api.getmetal.io/v1/data-entities/{id}"
description: "This endpoint deletes a specific data entity by its ID."
---

### Auth Headers

<ParamField required header="x-metal-api-key" type="string">
  An API key for your org.
</ParamField>

<ParamField required header="x-metal-client-id" type="string">
  A Client ID for your org.
</ParamField>

### Params

<ParamField required path="id" type="string">
  The unique ID of the data entity you wish to delete.
</ParamField>

<RequestExample>

```bash
# Example Request
curl -X DELETE "https://api.getmetal.io/v1/data-entities/{id}" \
--header 'x-metal-api-key: <api-key>' \
--header 'x-metal-client-id: <client-id>' \
```

</RequestExample>

<ResponseExample>

```json
204 No Content
```

</ResponseExample>
