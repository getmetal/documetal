---
title: "Get Indexes"
api: "GET https://api.getmetal.io/v1/apps/{appId}/indexes"
description: "This endpoint gets a list of indexes for an app."
---

### Cookie Header

<ParamField required header="token"  type="string">
  This is the users auth token.
</ParamField>

### Params

<ParamField required path="appId" type="string">
  The App ID
</ParamField>

### Response

<ResponseField name="data" type="Index Array">
  <Expandable title="properties">
    <ResponseField name="id" type="string">
      Id of the index
    </ResponseField>

    <ResponseField name="app" type="string">
      Id of the app of the index
    </ResponseField>

    <ResponseField name="name" type="string">
      Name of the index
    </ResponseField>

    <ResponseField name="model" type="string">
      Model used to generate the embeddings
    </ResponseField>

    <ResponseField name="dimensions" type="number">
      Dimensions of the embeddings
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request GET 'https://api.getmetal.io/v1/apps/{appId}/indexes' \
--header 'Content-Type: application/json' \
--header 'Authorization: Token <token>' \
```

</RequestExample>
