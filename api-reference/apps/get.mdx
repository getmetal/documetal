---
title: "Get Apps"
api: "GET https://api.getmetal.io/v1/apps"
description: "This endpoint gets a list of apps."
---

### Cookie Header

<ParamField required header="token"  type="string">
  This is the users auth token.
</ParamField>

### Response

<ResponseField name="data" type="object">
  Contains the object response.
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request POST 'https://api.getmetal.io/v1/apps' \
--header 'Content-Type: application/json' \
--header 'Authorization: Token <token>' \
```

</RequestExample>

<ResponseExample>

```json Response
{
  "data": {
    id: '1',
    name: 'My App'
  }
}
```

</ResponseExample>
