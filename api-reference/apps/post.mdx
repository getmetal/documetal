---
title: "Create App"
api: "POST https://api.getmetal.io/v1/apps"
description: "This endpoint creates an app."
---

### Cookie Header

<ParamField required header="token"  type="string">
  This is the users auth token.
</ParamField>

### Body

<ParamField body="name" type="string" placeholder="name of the index" required>
  Name of the App
</ParamField>


### Response

<ResponseField name="data" type="App Object">
  <Expandable title="properties">
    <ResponseField name="id" type="string">
      Id of the app
    </ResponseField>

    <ResponseField name="name" type="string">
      Name of the app
    </ResponseField>

    <ResponseField name="activeIndexId" type="string">
      Id of the active index
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request POST 'https://api.getmetal.io/v1/apps' \
--header 'Content-Type: application/json' \
--header 'Authorization: Token <token>' \
--data-raw '{
    "name": "Dave Mustaine's App"
}'
```

</RequestExample>

<ResponseExample>

```json Response
{
  "data": {
    "id": "1",
    "name": "Dave Mustaine"
  }
}
```

</ResponseExample>
