---
title: "API Key - All"
api: "GET https://api.getmetal.io/v1/keys"
description: "Retrieve all API keys for the authenticated organization"
---

### Auth Headers

<ParamField required header="x-metal-api-key" type="string">
  An API key for your organization.
</ParamField>

<ParamField required header="x-metal-client-id" type="string">
  A Client ID for your organization.
</ParamField>


<RequestExample>

```bash Example Request
curl --location --request GET 'https://api.getmetal.io/v1/keys' \
--header 'x-metal-api-key: <api-key>' \
--header 'x-metal-client-id: <client-id>' \
```

</RequestExample>



<ResponseExample>
```json
{
  "data": [
    {
      "id": "<key-id>",
      "name": "Example Key",
      "keyLastFour": "1234",
      "clientId": "<client-id>",
      "createdAt": "2023-08-29T18:00:00.000Z"
    },
    // ... more keys
  ]
}
```
</ResponseExample>
