---
title: "Get a Tuning"
api: "GET https://api.getmetal.io/v1/tunings/{tuningId}"
description: "This endpoint gets a single Tuning."
---

### Cookie Header

<ParamField required header="token" type="string">
  This is the users auth token.
</ParamField>

<ParamField required path="tuningId" type="string">
  The Tuning ID
</ParamField>

### Response

<ResponseField name="data" type="Tuning Object">
  <Expandable title="properties">
    <ResponseField name="id" type="string">
      Id of the Tuning
    </ResponseField>

    <ResponseField name="app" type="string">
      Id of the app for the Tuning
    </ResponseField>

    <ResponseField name="idA" type="string">
     Embedding ID A
    </ResponseField>

    <ResponseField name="idB" type="string">
      Embedding ID B
    </ResponseField>

    <ResponseField name="result" type="string">
      The score result. Either `-1` or `1`
    </ResponseField>

  </Expandable>
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request GET 'https://api.getmetal.io/v1/tunings/{tuningId}' \
--header 'Content-Type: application/json' \
--header 'Authorization: Token <token>' \
```

</RequestExample>

<ResponseExample>

```json Response
{
  "data": {

    "result": "1"
  }
}
```

</ResponseExample>
