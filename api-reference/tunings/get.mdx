---
title: "Get Tunings"
api: "GET https://api.getmetal.io/v1/apps/{appId}/tunings"
description: "This endpoint gets a list of tunings for an app."
---

### Auth Headers

<ParamField required header="x-metal-api-key" type="string">
  An API key for your org.
</ParamField>

<ParamField required header="x-metal-client-id" type="string">
  An Client ID for your org.
</ParamField>

### Params

<ParamField required path="appId" type="string">
  The App ID
</ParamField>

### Response

<ResponseField name="data" type="Tuning Array">
  <Expandable title="properties">
    <ResponseField name="id" type="string">
      Id of the Tuning
    </ResponseField>

    <ResponseField name="app" type="string">
      Id of the app for the Tuning
    </ResponseField>

    <ResponseField name="idA" type="string">
     Embedding ID A
    </ResponseField>

    <ResponseField name="idB" type="string">
      Embedding ID B
    </ResponseField>

    <ResponseField name="result" type="string">
      The score result. Either `-1` or `1`
    </ResponseField>

  </Expandable>
</ResponseField>

```bash Example Request
curl --location --request GET 'https://api.getmetal.io/v1/apps/{appId}/tunings' \
--header 'Content-Type: application/json' \
--header 'x-metal-api-key: <api-key>' \
--header 'x-metal-client-id: <client-id>' \
```

</RequestExample>
