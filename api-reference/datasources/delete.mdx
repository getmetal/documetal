---
title: "Delete Data Source"
api: "DELETE https://api.yourdomain.com/datasources/{id}"
description: "This endpoint deletes a data source by its ID."
---

### Auth Headers

<ParamField required header="Some-API-Key" type="string">
  An API key for your org.
</ParamField>

<ParamField required header="Some-Client-Id" type="string">
  A Client ID for your org.
</ParamField>

### Params

<ParamField required path="id" type="string">
  The unique identifier of the data source you wish to delete.
</ParamField>

<RequestExample>

```bash
# Example Request
curl -X DELETE "https://api.yourdomain.com/datasources/some_id" \
--header 'Content-Type: application/json' \
--header 'Some-API-Key: <api-key>' \
--header 'Some-Client-Id: <client-id>'
```
</RequestExample>

<ResponseExample>

```json Response
{
  "message": "Data source successfully deleted."
}
```
</ResponseExample>
