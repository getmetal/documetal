### Cookie Header

<ParamField required header="token"  type="string">
  This is the users auth token.
</ParamField>

### Response

</ResponseField>

<RequestExample>

```bash Example Request
--header 'Content-Type: application/json' \
--header 'Authorization: Token <token>' \
```

</RequestExample>
