---
title: "Get Keys"
api: "GET https://api.getmetal.io/v1/keys"
description: "This endpoint gets a list of key."
---

### Cookie Header

<ParamField required header="token" type="string">
  This is the users auth token.
</ParamField>

### Response

<ResponseField name="data" type="Key Array">
  <Expandable title="properties">
    <ResponseField name="id" type="string">
      ID of the key
    </ResponseField>

    <ResponseField name="name" type="string">
      Name of the key
    </ResponseField>

    <ResponseField name="keyLastFour" type="string">
     Last 4 digits of the key
    </ResponseField>

    <ResponseField name="clientId" type="string">
     Client ID of the key
    </ResponseField>

  </Expandable>
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request POST 'https://api.getmetal.io/v1/keys' \
--header 'Content-Type: application/json' \
--header 'Authorization: Token <token>' \
```

</RequestExample>

<ResponseExample>

{
  "data": [
