## Install

```bash
npm install @getmetal/metal-sdk
# or
yarn add @getmetal/metal-sdk
```

## Modules

```ts
import { Metal, Motorhead } from "@getmetal/metal-sdk";

const API_KEY = "pk_123";
const CLIENT_ID = "ci_123";

// Metal is the retrieval module
const metalRetriever = new Metal(API_KEY, CLIENT_ID, "123");

// Motorhead is the memory module
const motorMemory = new Motorhead(API_KEY, CLIENT_ID);
```
